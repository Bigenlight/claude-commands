현재 작업 디렉토리의 CLAUDE.md와 README를 읽고, 이 리포의 전체 흐름과 컨텍스트를 파악해.

## 수행 순서

1. **CLAUDE.md 읽기** — 프로젝트 루트의 `CLAUDE.md` 를 Read로 읽기
2. **README 찾기** — `README.md` 또는 `README` 가 있으면 읽기 (없으면 스킵)
3. **최근 git 로그** — `git log --oneline -10` 으로 최근 작업 흐름 확인
4. **폴더 구조 확인** — 루트 디렉토리 목록 (`ls` 또는 Glob `*`) 으로 현재 폴더 구성 파악

## 출력 형식

읽은 내용을 바탕으로 아래 항목을 **간결하게** 정리해서 출력:

- **리포 목적**: 한 줄 요약
- **폴더 구조**: 주요 폴더 + 용도 (CLAUDE.md에 이미 있으면 그대로 인용)
- **현재 활성 작업**: git 로그 기반으로 최근에 뭘 하고 있었는지
- **주요 컨벤션**: 알아두면 유용한 것들 (노트 스타일, frontmatter 규칙 등)
- **지금 뭐부터 보면 좋을지**: 현 상태 기반으로 간단한 진입점 제안

출력은 한국어로, 구어체 반말. 기술 용어는 영어 유지.
